## 📌 Tentang Project
Aplikasi Manajemen Perpustakaan ini dibangun menggunakan **Laravel 8** untuk memenuhi kebutuhan Uji Kompetensi Keahlian (UKOM) PPLG. Sistem ini mencakup pengelolaan data buku (dengan fitur arsip), manajemen anggota, dan transaksi peminjaman.
